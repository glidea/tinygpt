# TinyGPT 🤖

> TinyGPT 旨在帮助非 AI 领域的开发者轻松理解大语言模型的核心原理

---

### ✨ 特性

*   **完整训练流程**: 涵盖从 **预训练、SFT 到 DPO**
*   **低门槛**: 无需云端 GPU，同时兼容 Mac M 芯片
*   **为学习而生**:
      * 内置 **[@teacher AI](/teacher.md)**，引导式教学
      * 不追求性能和效果，仅使用经典算法做入门实现
      * 有类型注释，懂的都懂 🤷🏻‍♀️...


### 🚀 开始吧！

> 每个章节相互独立，可任意开始

也可直接让老师带你，比如：

```bash
# 在 Cursor，Claude Code 或任何带 Bash Tool 的环境中：
```
